Welcome to PHP Oxygen

This page is displayed because you have to configure PHP Oxygen to redirect to a default action in a module.

You must define a default action to launch. For that you must have a default route configured in routes.xml. The file must be located here :

{$routeFilePath}

If not, you have two possibilities :

  • Create routes.xml in the webapp/config folder.
  • Create a module with a routes.xml file in the config folder. PHP Oxygen will automaticaly parse all routes.xml files in modules and build the file in webapp/config

In the xml file, the default route seems like this :

<?xml version="1.0" encoding="UTF-8"?>
<routes>
    <route rule="default" redirect="module/action" />
</routes>

For additional details please read the user guide
