Pré-requis

Serveur web

  • Un serveur Apache (version >= 2.0)

Module Apache requis

  • rewrite

Modules Apache recommandés

  • deflate
  • expires
  • headers

Base de données

  • MySQL (version >= 5.1)

PHP

  • Version >= 5.2.0 requis (>= 5.3.0 recommandé)

Extensions PHP requises

  • gd
  • mbstring
  • pdo

Extensions PHP recommandées

  • APC
  • memcache
  • mcrypt
