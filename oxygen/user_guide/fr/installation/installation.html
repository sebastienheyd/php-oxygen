Installation

Pour installer PHP-Oxygen, il suffit de récupérer et de placer correctement les fichiers dans votre espace d'hébergement.

IMPORTANT ! Dossier www

Pour fonctionner, votre hébergement doit être configuré de façon à ce que le dossier www inclus dans PHP-Oxygen soit le dossier public. Tous les autres dossier sont donc au même niveaux que ce dernier et vous devez configurer le vhost pour que www soit le seul dossier accessible.

Exemple :

<VirtualHost *:80> Servername example.com DocumentRoot /var/example.com/www </VirtualHost>

Dans cet exemple, le dossier /var/example.com contient les fichiers de PHP-Oxygen (racine) et /var/example.com est le dossier public (dossier www de PHP-Oxygen).

Pour plus de détails sur les dossiers de PHP-Oxygen, voir la rubrique Structure / squelette

Récupération des fichiers

Via SVN

Il est possible de récupérer facilement la version de développement de PHP-Oxygen dans le dossier courant via SVN en utilisant la commande suivante :

svn checkout http://php-oxygen.googlecode.com/svn/trunk/ .
