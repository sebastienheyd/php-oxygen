Error

La classe Error permet la gestion des erreurs mais également leur personnalisation.

Méthodes

show401()

Méthode statique qui stoppe le script courant et affiche une erreur 401.

Error::show401();
Le gabarit se trouve dans le dossier oxygen/errors. Ce dernier peut être surchargé en le recréant dans webapp/hooks/errors.

show404()

Méthode statique qui stoppe le script courant et affiche une erreur 404.

Error::show404();
Le gabarit se trouve dans le dossier oxygen/errors. Ce dernier peut être surchargé en le recréant dans webapp/hooks/errors.
