Classe Config

La classe Config permet de lire le contenu du fichier de configuration de l'application.

Méthodes

get()

Méthode statique. Permet de récupérer la valeur d'une variable de configuration ou d'une section sinon renvoie la valeur par défaut (false si non spécifiée).

// Retourne la valeur error_level de la section debug, sinon 'off' Config::get('debug.error_level', 'off'); // Retourne un tableau associatif de la section debug Config::get('debug');

getEnvironment()

Méthode statique. Récupère le nom d'environnement courant sinon "default". Voir fichier de configuration.
