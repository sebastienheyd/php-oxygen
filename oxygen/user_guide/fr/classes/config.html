Classe Config

La classe Config permet de lire le contenu du/des fichier(s) de configuration de l'application.

Emplacement

Le(s) fichier(s) de configuration se trouve(nt) dans le dossier webapp/config.

Contextualisation

La configuration de PHP Oxygen est contextualisée. Cela veut dire que vous pouvez avoir un fichier de configuration par environnement. Cela est très utile dans le cas ou vous développez sur un ou plusieurs serveurs et que vous avez un ou plusieurs serveurs de production par exemple.
Pour cela il suffit de définir la variable d'environnement APP_ENV soit dans le fichier .htaccess, soit directement dans la configuration du vhost Apache. Par exemple :

SetEnv APP_ENV "development"

Dans cet exemple, le fichier de configuration utilisé sera webapp/config/development.ini
